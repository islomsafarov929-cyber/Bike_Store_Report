# Bike Store Report рЯЪівАНвЩВпЄПрЯУК
This project contains an analytical report on bike store sales.
The report was built using Power BI and uploaded to Microsoft Fabric, providing insights into:
- Sales volume and growth trends
- Analysis by bike brand, category and model
- Store distribution of sales
- Pricing and profitability metrics
View the Dashboard
You can explore the full interactive dashboard here:
https://app.fabric.microsoft.com/groups/me/reports/ab693536-fae0-40f0-8cf6-63bec2703262/a4c7d3e542e9ec4004ac?experience=power-bi

Purpose
The goal of this project is to support business decision-making in the cycling industry by identifying sales trends, highlighting key performance metrics, and delivering actionable insights for store management and strategy.
